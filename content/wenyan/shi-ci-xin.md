# 高频实词“信”用法整理

“信”是高中文言文中常见实词，考试中常结合语境考查其不同义项。

## 常见义项

| 义项 | 说明 |
| --- | --- |
| 诚信 | 言而有信，守信用 |
| 相信 | 信任、相信某人或某事 |
| 确实 | 表示真实、的确 |
| 随意 | 与“信步”等词相关 |

## 记忆提示

遇到“信”字，不要先套固定意思，要先看它与前后词语的关系，再结合句意判断。

> 文言实词的学习，重在语境迁移，而不只是背一组解释。
