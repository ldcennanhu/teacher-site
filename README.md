# 高中语文教学资料馆

这是一个纯静态 HTML + CSS 个人教学官网，可直接部署到 GitHub Pages、Netlify、Vercel 或任意静态空间。

## 文件结构

- `index.html`：网站首页，六大主栏目卡片在这里修改。
- `css/styles.css`：全站统一样式，包含配色、布局、响应式适配。
- `pages/`：六个二级栏目页面，可继续复制新增页面。
- `assets/`：图片与后续可替换的素材资源。

## 新增栏目方法

1. 复制 `pages/resources.html`，改成新的页面文件名。
2. 修改页面里的标题、导航高亮、栏目小节和正文内容。
3. 在 `index.html` 的 `.column-grid` 中复制一张栏目卡片，修改链接与文字。
4. 如需加入顶部导航，在所有 HTML 文件的 `.nav-links` 中增加对应链接。

## 修改风格

主要颜色在 `css/styles.css` 顶部的 `:root` 中：

- `--qing`：浅青色
- `--rice`：米白色
- `--ink`：深灰正文色
- `--qing-deep`：导航、按钮与强调色

保持低饱和度、大留白和清晰层级，适合长期作为教师个人公益资料站维护。
